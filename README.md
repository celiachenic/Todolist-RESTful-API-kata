# Todolist RESTful API kata
使用 Node.js 原生 HTTP 模組實作 Todo List RESTful API。

於課程後自主重構與重複練習，目標是在不依賴框架的情況下熟悉 RESTful API 開發流程、路由設計、資料驗證與錯誤處理。
## Demo
